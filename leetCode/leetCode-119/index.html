leetCode-119:Pascal's Triangle II

发表于 2023-04-09 分类于 leetCode

问题描述

给定一个正整数 n，要求找出杨辉三角中第 n 行的数字列表。题目链接：点我

样例输入输出

输入：3

输出：[1,3,3,1]

输入：0

输出：[1]

问题解法

观察杨辉三角每一行的数字，可以发现其数字呈现以下规律：1、n、n(n-1)/2、n(n-1)(n-2)/(2*3)、n(n-1)(n-2)(n-3)/(2*3*4)

、…、1，所以可以根据此直接算出每一行中的每个数字。代码如下
