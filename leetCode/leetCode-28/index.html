leetCode-28:Implement strStr()

发表于 2018-05-27 更新于 2018-05-30 分类于 leetCode

问题描述

给出一个待匹配的字符串A和匹配的字符串B，要求判断B是否是A的子字符串。如果是则输出匹配的A字符串的起始下标，如果不是子字符串则输出-1。

注意：如果B字符串是空的，返回 0 ，这与String.indexOf函数效果相同
